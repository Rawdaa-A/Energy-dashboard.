# Energy Consumption & Pricing Analysis Dashboard ⚡🔋

## 📝 Project Overview
This project focuses on analyzing energy data to understand consumption patterns and pricing trends. Using **Microsoft Excel's advanced features**, I transformed raw energy data into a two-page interactive dashboard that facilitates data-driven decision-making.

## 🚀 Key Features
The dashboard consists of two specialized pages:

1. **Energy Consumption Dashboard:** * Analyzes energy usage over time.
   * Visualizes consumption patterns using line charts and Key Performance Indicators (KPIs).
   
2. **Energy Price Dashboard:** * Tracks energy price fluctuations and market trends.
   * Includes interactive **Slicers** for deep diving into specific timeframes or categories.

## 🛠️ Data Workflow (Technical Steps)
* **ETL with Power Query:** Imported and cleaned the energy datasets, ensuring consistency and accuracy.
* **Data Modeling:** Built a robust data model using **Power Pivot** to link multiple tables.
* **Analysis:** Created Pivot Tables and calculated measures to extract meaningful insights.
* **Visualization:** Designed an intuitive UI with interactive filters and dynamic charts.
